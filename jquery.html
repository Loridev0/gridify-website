Was ist jQuery?

jQuery ist eine schnelle, kleine und funktionsreiche JavaScript-Bibliothek. Sie macht Dinge wie HTML-Dokumentnavigation und -Manipulation, Event-Handling, Animation und Ajax viel einfacher mit einer einfach zu bedienenden API, die in vielen Browsern funktioniert.

Vereinfachte Syntax

Schreibe weniger, erreiche mehr mit jQuery's eleganter Syntax.

Cross-Browser Kompatibilität

Funktioniert zuverlässig in allen gängigen Browsern.

Leistungsstark & Erweiterbar

Umfangreiche Funktionen und erweiterbar durch Plugins.

Wie man jQuery und verwandte Bibliotheken verwendet

Schritt 1: Notwendige Skripte einbinden

Füge die folgenden Script-Tags am Ende des <body> deiner HTML-Datei ein:

<!-- jQuery (Grundlage für viele JavaScript-Funktionalitäten) -->
<script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>

<!-- Popper.js (wird für Bootstrap-Komponenten benötigt) -->
<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.11.6/dist/umd/popper.min.js"></script>

<!-- Bootstrap JS (für interaktive Bootstrap-Komponenten) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

<!-- Lightbox JS (für die Bildergalerie-Funktionalität) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/lightbox2/2.11.3/js/lightbox.min.js"></script>

Schritt 2: Dokument bereit machen

Stelle sicher, dass dein Code erst ausgeführt wird, wenn das Dokument vollständig geladen ist:

$(document).ready(function() {
    // Dein jQuery-Code hier
});

Oder kürzer:

$(function() {
    // Dein jQuery-Code hier
});

Schritt 3: Elemente auswählen und manipulieren

Verwende jQuery-Selektoren, um HTML-Elemente auszuwählen und zu manipulieren:

// Nach ID auswählen und Text ändern
$("#myElement").text("Neuer Text");

// Nach Klasse auswählen und CSS ändern
$(".myClass").css("color", "red");

// Event-Handler hinzufügen
$("button").click(function() {
    alert("Button geklickt!");
});
